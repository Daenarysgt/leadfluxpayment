Os sons precisam ser adicionados manualmente na pasta public/sounds

Arquivos de som necessários para o componente de notificação:

- sale.mp3 - Som de venda (ex: som de caixa registradora)
- success.mp3 - Som de sucesso (ex: som de confirmação, "ping" positivo)
- alert.mp3 - Som de alerta (ex: notificação de atenção)
- notification.mp3 - Som de notificação geral

Recomendações:
1. Use arquivos MP3 curtos (1-3 segundos) para melhor experiência do usuário
2. Certifique-se que o volume dos arquivos seja adequado (não muito alto)
3. Arquivo deve ter formato .mp3 e ter o nome exato conforme listado acima
4. Tamanho recomendado: menos de 100KB por arquivo de som

Como adicionar os sons:
1. Baixe os arquivos de som desejados
2. Renomeie-os para corresponder aos nomes acima (ex: sale.mp3)
3. Coloque-os na pasta public/sounds/ 
4. O sistema buscará os arquivos automaticamente quando a notificação for ativada

Observação: Para testar os sons, use a função "Testar Som" no painel de configuração da notificação.
